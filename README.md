# 🔐 Keysmith – Password Generator

Keysmith is a modern, responsive password generator built with **React** and **Vite**. It helps users generate strong, secure, and customizable passwords instantly while keeping everything completely local in the browser.

## ✨ Features

- 🔒 Generate strong and secure passwords
- 📏 Adjustable password length
- 🔠 Include Uppercase (A–Z)
- 🔡 Include Lowercase (a–z)
- 🔢 Include Numbers (0–9)
- 🔣 Include Special Symbols (!@#$...)
- 🚫 Exclude confusing characters (look-alikes)
- 📋 One-click copy to clipboard
- 🔄 Instant password regeneration
- 📊 Password strength indicator
- 🌙 Modern responsive UI
- 🛡️ 100% Client-side – Nothing leaves your browser

## 🛠️ Tech Stack

- React.js
- Vite
- JavaScript (ES6+)
- CSS
